## Installation

```
$ npm i skip-navigation
```

## Usage

```html
<!DOCTYPE html>
<html>
  <body>
    <skip-button></skip-button>
    <a href="#">Irrelevant link 1</a>
    <a href="#">Irrelevant link 2</a>

    <!-- More irrelevant links.. -->

    <main>
        <skip-anchor></skip-anchor>

        <!-- Main content -->
    </main>
  </body>
</html>
```
